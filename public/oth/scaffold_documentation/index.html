General

Scaffold is a HTML site template built with twitter bootstrap and designed with metro style. This template coming with single custom color : 'orange' which stored in file 'orange.css' in folder assets/color

We've separated the custom color (orange) in a single file : orange.css to make you easy to understand and customize or overwrite with your own color/style

Since Scaffold tempalte built with bootstrap so you can go to http://twitter.github.com/bootstrap/ to learn more about twitter boostrap. Also lot of basic documentation or template overview already added on each page under 'Features' menu.

Since it's HTML CSS based you can edit all text, content with HTML editor like notepad++ or anykind else. Quite easy for people with basic HTML-CSS knowledge :)

Responsive technology

Scaffold template using 'bootstrap-responsive.css' to make the template responsive on various screen sizes.

 

Media queries:


/* Large desktop */
@media (min-width: 1200px) { ... }

/* Portrait tablet to landscape and desktop */
@media (min-width: 768px) and (max-width: 979px) { ... }

/* Landscape phone to portrait tablet */
@media (max-width: 767px) { ... }

/* Landscape phones and down */
@media (max-width: 480px) { ... }

LOGO

The current logo image located in folder 'assets/img/logo.png'. You can change it with your own logo by replacing current logo, make sure use same fine name and format ( logo.png ) so you don't need to edit the style.css file. Current logo size is 200 x 40 (width - height in pixel).

HTML code for the logo:

          <!-- logo -->
          <a class="brand logo" href="index.html">
            <img src="assets/img/logo.png" alt="" />
          </a>
          <!-- end logo -->
